 SheCodes React Session – useEffect Demo

This project is a simple React application built for a **SheCodes React session** to demonstrate how the `useEffect` hook works.

---

## 📌 Project Overview

The app fetches user data from a public API and displays it in a clean, responsive card layout.

It is designed to help beginners understand:
- How to use `useEffect` for data fetching
- How to manage state using `useState`
- How to render dynamic data in React
- Basic UI styling with grid layout

---

## 🚀 Features

- Fetches data from a public API
- Displays users in responsive cards (not a list)
- Shows a loading state while fetching data
- Simple hover effect on cards
- Beginner-friendly and easy to understand

---

## 🧠 Concepts Covered

- React Functional Components
- `useState` for state management
- `useEffect` for side effects (API calls)
- Conditional rendering
- Mapping over arrays to display data

---

## 🌐 API Used

- https://jsonplaceholder.typicode.com/users
